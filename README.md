# 302FinalProject
Final Project for Dr. Emrich's COSC302 class

Reflex Racer
2D racing game made in Unity. Player opens up game, greeted by main menu with two (three?) options, to play the game or view the leaderboard (or see credits?). When playing the game the player is counted down from 3 to which the race starts. During the race the player is randomly prompted 4-5 times to 'switch gears' and their reaction time after this message pops up is what is tracked. After each gear change, the lead car is adjusted based on whether the reaction was slow or fast. After the race is over, all individual reaction times are added together for a total which is displayed along with a win/loss pop up. This time is also added to the leaderboard (if it was quick enough). Prompt the player to replay, view leaderboard, or return to menu after each race. Leaderboard displays top ~25 times maybe more/less.
